# Instruction Set (TombVM v0.1)

| Opcode | Instruction        | Description                 |
|--------|--------------------|-----------------------------|
| 0x01   | MOV R, value       | Set register to value       |
| 0x02   | ADD R, value       | Add value to register       |
| 0x03   | PRINT R            | Print register value        |
| 0xFF   | HALT               | Stop execution              |
